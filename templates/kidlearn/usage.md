# KidLearn Template Usage

## How To Use

Use this template for WeChat articles about children's learning products, AI education tools, parenting-facing learning guides, and product design notes.

Recommended flow:

1. Start with a chunky title card.
2. Explain the problem in simple parent-friendly language.
3. Use one reward insight card for the main claim.
4. Use blue learning tips for practical advice.
5. Use yellow gentle warnings for risks.
6. End with a checklist or one memorable question.

## WeChat Compatibility

Rich-text safe:

- Title card.
- Learning tip.
- Reward insight card.
- Gentle warning.
- Checklist.
- Closing CTA.

Image-based:

- Cover.
- Reward poster.
- Activity map.
- Complex lesson path.
- Dense comparison matrix.

Verification-needed:

- Multi-column cards.
- Nested cards.
- Large shadowed blocks after pasting into the WeChat editor.

## Paste-Test Checklist

- Check that the yellow title card keeps its background.
- Check that the red title and blue pill remain readable.
- Check that shadows do not create awkward spacing.
- Check mobile preview before publishing.
- If a component breaks, convert it into an image module.
